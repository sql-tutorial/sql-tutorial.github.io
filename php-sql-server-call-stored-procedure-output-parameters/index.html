PHP SQL Server: вызов хранимой процедуры с параметрами OUTPUT

Время чтения 4 мин.Опубликовано 15.04.2025

Это руководство начинается с того места, где закончился урок по вызову хранимой процедуры из PHP.

В этом уроке вы узнаете, как вызвать хранимую процедуру с параметрами OUTPUT в PHP.

Создание хранимой процедуры

Шаг 1. Запустите Microsoft SQL Server Management Studio(SSMS) и подключитесь к SQL Server.

Шаг 2. Выполните следующий оператор, чтобы создать новую хранимую процедуру с именем createOrder с выходным параметром: 

CREATE PROCEDURE CreateOrder
    @CustomerId INT,
    @BookId INT,
    @Quantity INT,
    @Price DECIMAL(18, 2),
    @OrderDate DATE,
    @OrderId INT OUTPUT
AS
BEGIN
    SET NOCOUNT ON;
    BEGIN TRY
        -- Start a transaction
        BEGIN TRANSACTION;
        -- Check inventory
        DECLARE @CurrentQty INT;
        SELECT @CurrentQty = Qty FROM Inventories WHERE BookId = @BookId;
        IF @CurrentQty IS NULL OR @CurrentQty  0
        BEGIN
            ROLLBACK TRANSACTION;
        END
        -- Return error information
        DECLARE @ErrorMessage NVARCHAR(4000);
        DECLARE @ErrorSeverity INT;
        DECLARE @ErrorState INT;
        SELECT @ErrorMessage = ERROR_MESSAGE(),
               @ErrorSeverity = ERROR_SEVERITY(),
               @ErrorState = ERROR_STATE();
        RAISERROR(@ErrorMessage, @ErrorSeverity, @ErrorState);
    END CATCH
END

    public function create($customerId, $bookId, $quantity, $price, $orderDate)
    {
        try {
            // Prepare the stored procedure call
            $sql = "DECLARE @OrderId INT;
                    EXEC CreateOrder :customerId, :bookId, :quantity, :price, :orderDate, @OrderId OUTPUT;
                    SELECT @OrderId AS OrderId;";
            $stmt = $this->conn->prepare($sql);
            // Bind parameters
            $stmt->bindParam(':customerId', $customerId, PDO::PARAM_INT);
            $stmt->bindParam(':bookId', $bookId, PDO::PARAM_INT);
            $stmt->bindParam(':quantity', $quantity, PDO::PARAM_INT);
            $stmt->bindParam(':price', $price, PDO::PARAM_STR);  // Using PDO::PARAM_STR for DECIMAL
            $stmt->bindParam(':orderDate', $orderDate, PDO::PARAM_STR);
            // Execute the stored procedure
            $stmt->execute();
            // Fetch the order ID from the result set
            $result = $stmt->fetch(PDO::FETCH_ASSOC);
            // Return the order ID
            return $result['OrderId'];
        } catch(PDOException $e) {
            throw new Exception("Failed to create order: " . $e->getMessage());
        }
    }
}

Запуск приложения

Шаг 1. Измените index.php, который использует метод createOrder объекта OrderDB: 

create(
    $customerId,
    $bookId,
    $quantity,
    $price,
    $orderDate
);
echo "New Order ID: " . $orderId;

Как это работает.

Сначала загрузите файл orderdb.php, чтобы использовать класс OrderDB. 

require_once 'orderdb.php';

Во-вторых, подключитесь к SQL Server и назначьте возвращенный объект PDO переменной $conn: 

$conn = require_once 'connect.php';

В-третьих, создайте новый объект OrderDB с объектом PDO: 

$orderDB = new OrderDB($conn);

В-четвертых, инициализируем данные заказа: 

$customerId = 1;
$bookId = 1;
$quantity = 10;
$price = 9.99;
$orderDate = '2024-07-31';

В-четвертых, вызовите метод create() для создания нового заказа: 

$orderId = $orderDB->create(
    $customerId,
    $bookId,
    $quantity,
    $price,
    $orderDate
);

Наконец, отобразите идентификатор заказа: 

echo "New Order ID: " . $orderId;

Шаг 2. Запустите файл index.php в веб-браузере, вывод будет выглядеть следующим образом: 

New Order ID: 2
