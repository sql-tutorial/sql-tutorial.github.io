SQL Server СОЗДАТЬ БАЗУ ДАННЫХ

Время чтения 2 мин.Опубликовано 15.04.2025

Краткое описание: в этом руководстве вы узнаете, как создать новую базу данных в SQL Server с помощью оператора CREATE DATABASE или SQL Server Management Studio.

Создание новой базы данных с помощью оператора CREATE DATABASE

Оператор CREATE DATABASE создает новую базу данных. Ниже показан минимальный синтаксис оператора CREATE DATABASE: 

CREATE DATABASE database_name;

В этом синтаксисе вы указываете имя базы данных после ключевого слова CREATE DATABASE.

Имя базы данных должно быть уникальным в пределах экземпляра SQL Server. Оно также должно соответствовать правилам идентификатора SQL Server. Обычно имя базы данных содержит максимум 128 символов.

Следующий оператор создает новую базу данных с именем TestDb: 

CREATE DATABASE TestDb;

После успешного выполнения оператора вы можете просмотреть вновь созданную базу данных в Object Explorer. Если новая база данных не отображается, вы можете нажать кнопку Refresh или клавишу F5 на клавиатуре, чтобы обновить список объектов.

В этом операторе перечислены все базы данных на SQL Server: 

SELECT 
    name
FROM 
    master.sys.databases
ORDER BY 
    name;

Или вы можете выполнить хранимую процедуру sp_databases: 

EXEC sp_databases;

Создание новой базы данных с помощью SQL Server Management Studio

Сначала щелкните правой кнопкой мыши базу данных и выберите пункт меню «Новая база данных…».

Во-вторых, введите имя базы данных, например, SampleDb, и нажмите кнопку «ОК».
